Project: Portfolio and Investment Analysis (AMZN & COKE)
Overview
This project demonstrates the application of quantitative methods to evaluate financial assets. Using historical data from Yahoo Finance, I analyzed the risk-return profiles of Amazon (AMZN) and Coca-Cola Consolidated (COKE) for the period of 2025-2026.

Quantitative & Digital Skills Demonstrated:
Data Preparation: Extraction, cleaning, and chronological structuring of financial datasets from Yahoo Finance.

Statistical Analysis: Calculation of Mean Returns, Standard Deviation (Risk), and Correlation Coefficients to assess asset performance.

Financial Modeling: Applied Modern Portfolio Theory (MPT) concepts to evaluate diversification efficiency.

Digital Tools: Proficiency in Google Sheets/Excel for financial calculations and GitHub for project version control and documentation.

Key Analytical Results:
Correlation Coefficient: 0.8907 (Strong positive correlation).

Risk (Standard Deviation): AMZN (0.1435) | COKE (0.1437).

Key Insight: The high correlation suggests that both assets are driven by similar market factors. This highlights the importance of cross-sector diversification to reduce unsystematic risk.
